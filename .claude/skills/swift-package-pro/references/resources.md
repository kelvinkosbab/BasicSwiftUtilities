# Resources in Swift Packages

## `.process` vs `.copy`

In `Package.swift`, declare resources with one of two strategies:

- **`.process(_:)`** — SPM applies platform-appropriate processing. Asset catalogs become compiled `.car` files, storyboards become `.nib`, strings files are validated, etc.
- **`.copy(_:)`** — SPM copies the file verbatim. Use for files that should not be transformed.

Default to `.process` unless you need raw access to the file as it exists on disk.

```swift
.testTarget(
    name: "CoreStorageTests",
    dependencies: ["CoreStorage"],
    path: "CoreStorage/Tests",
    resources: [.process("Resources")]
)
```

## Accessing resources at runtime

Use `Bundle.module` (auto-generated by SPM):

```swift
let url = Bundle.module.url(forResource: "data", withExtension: "json")!
let data = try Data(contentsOf: url)
```

`Bundle.module` is only available in modules that declare resources. Otherwise the symbol won't be generated.

## SPM limitation: `.xcdatamodeld`

**SPM cannot compile `.xcdatamodeld` to `.momd` bundles.** Only Xcode (`xcodebuild`) can.

This means:

- `swift build` and `swift test` will succeed but `Bundle.module` won't contain the compiled `.momd`.
- Loading the model with `NSManagedObjectModel(contentsOf: Bundle.module.url(forResource: "MyModel", withExtension: "momd")!)` returns `nil`.
- Tests that use the model via `Bundle.module` will fail under `swift test`.

**Workarounds:**

1. **Build the model programmatically** for tests (preferred for portability):

   ```swift
   func makeMyModel() -> NSManagedObjectModel {
       let entity = NSEntityDescription()
       entity.name = "MyEntity"
       entity.managedObjectClassName = "MyEntity"
       // ... add attributes ...
       let model = NSManagedObjectModel()
       model.entities = [entity]
       return model
   }
   ```

2. **Run tests via `xcodebuild test`** instead of `swift test` — Xcode will compile the `.xcdatamodeld`.

## Localized strings

`.xcstrings` (String Catalogs) are processed by SPM. Use `String(localized:bundle:)` with `Bundle.module`:

```swift
let title = String(localized: "welcome.title", bundle: .module)
```

## Excluding files

If a directory contains files that aren't sources or resources, exclude them:

```swift
.target(
    name: "MyModule",
    path: "MyModule/Sources",
    exclude: ["README.md", "Notes.txt"]
)
```

Otherwise SPM may complain about unhandled files.
